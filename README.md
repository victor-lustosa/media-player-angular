# Poc Media Player

Este projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) versão 15.2.11.

## Sobre este projeto

Este é um projeto de prova de conceito (POC) desenvolvido para testar a funcionalidade de media player com Angular 15. A aplicação suporta reprodução de múltiplos formatos de áudio e vídeo:

### Formatos de Áudio Suportados
- MP3
- WAV
- OGG
- FLAC
- AAC
- MP4

### Formatos de Vídeo Suportados
- MP4
- MKV
- AVI
- WMV
